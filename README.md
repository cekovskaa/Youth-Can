# Youth-Can Organization Website Project

## Project Overview

A web-based platform for the Association of citizens Youth Can/ Здружение на граѓани Младите можат, this is a non-profit and independent civic organization that aims to encourage young people to take an active part in the development of a democratic society, through active citizenship and social activism.

The project is built using HTML, CSS, Javascript and it contains features for form validation and responsive design to ensure optimal user experience across different devices.

## Techologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Git/GitLab for version control

## Features

- **Responsive design using Bootstrap**
- **Interactive form with client-side validation**
- **Dynamic image carousel showcasing key content**
- **Custom hover effects for enhanced user interaction**
- **Mobile-friendly layout**
